# Book-Search-Engine



## Description
The Book search engine is an web application build with MERN stack. It allows users to search book from google book API where user can save or remove it for later use.


## Core Skills
* JavaScript
* MongoDB
* Express.js
* React
* Node.js
* Bcrypt
* JWT
